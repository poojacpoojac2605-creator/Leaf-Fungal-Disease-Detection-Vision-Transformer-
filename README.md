Overview

This project focuses on detecting plant leaf fungal diseases using a Vision Transformer (ViT) model. It improves accuracy by capturing both global and local image features using self-attention mechanisms.

🚀 Features
Automated plant disease detection
Uses Vision Transformer (ViT) architecture
High accuracy compared to traditional CNN models
Evaluation using precision, recall, F1-score, confusion matrix

🛠️ Tech Stack
Python
TensorFlow / PyTorch
OpenCV
Deep Learning

⚙️ How It Works
Collect and preprocess leaf image dataset
Apply image augmentation techniques
Train Vision Transformer model
Evaluate performance using metrics
Predict disease from input leaf image
📊 Results

Improved accuracy over CNN models
Better feature extraction using self-attention
